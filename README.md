단순한 키보드 자동 입력 매크로

최대 6개의 키 입력 가능
키 입력 지연 시간 및 반복 간격 설정 가능
설정은 실행파일명.xml 로 실행파일 위치에 저장 됨
핫 키를 A-Z 중 직접 설정 가능
핫 키가 실행 된 프로그램으로만 키 입력이 됨
ㄴ PostMessage: 해당 프로그램으로 키 입력을 보냄
ㄴ SendKeys: 해당 프로그램일 때만 키 입력이 됨

기타: 초보라서 코드가 효율적인지는 모르겠습니다.
